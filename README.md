# Conversor de Temperatura

Programa que converte temperaturas entre **Celsius**, **Fahrenheit** e **Kelvin**.

## GitHub Actions

O workflow em `.github/workflows/tests.yml` roda os testes a cada commit em uma matriz:

| Sistema operacional | Versões de Python |
|---------------------|-------------------|
| Ubuntu              | 3.9, 3.11, 3.12   |
| macOS               | 3.9, 3.11, 3.12   |
| Windows             | 3.9, 3.11, 3.12   |
